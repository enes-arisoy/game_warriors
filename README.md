# 🎮 Game Warriors

A lightweight gaming landing page built with HTML, CSS, and JavaScript. This project includes a responsive header, hero section, game cards, news sections, and a mobile menu interaction.

## 🚀 Features

- Responsive navigation menu with hamburger toggle
- Animated mobile menu support
- Clean landing page layout with hero and latest news sections
- Game showcase cards with badge styles
- Modern dark theme with accent colors

## 💡 Notes

- The mobile menu toggle uses the `open` class to show/hide the nav list.
- Make sure your `style.css` contains the mobile styles and `.open` selector for the navigation.
